# Haizi_Poem_Explorer
Using Python to analyze the poems from Haizi

2019年的夏天，我从茶卡盐湖开车向西，目的地是德令哈。那天晚上，公路两边是牧场，高速上车很少，只有大量蚊虫围着车灯飞。

去德令哈住一晚，是因为海子那首写于当地的《日记》（1988.6.8—10）。

>姐姐，今夜我在德令哈，夜色笼罩
>
>姐姐，我今夜只有戈壁
>
>草原尽头我两手空空
>
>悲痛时握不住一颗泪滴
>
>姐姐，今夜我在德令哈
>
>这是雨水中一座荒凉的城
>
>除了那些路过的和居住的
>
>德令哈……今夜
>
>这是唯一的，最后的，抒情。
>
>这是唯一的，最后的，草原。


第二天早晨下着小雨，我在城里逛了逛。四周还是戈壁和草原，但德令哈已经不是诗中那座荒凉的城——作为海西州首府，它现在挺繁华，也比较现代化。不过当地人没有忘记海子，给他建了一座诗歌陈列馆。

海子去世时，朋友们担心他会被人遗忘。现在看来这种担心是多余的。他的诗被选入语文课本，“面朝大海，春暖花开”几乎人人皆知。

我手头有西川编的《海子诗全集》，经常翻看。除了感叹海子的文字细腻，我也好奇他最喜欢用哪些词。于是用 Python 爬了海子诗歌的文本，用 jieba 分词统计词频，按频率排序，写成这篇文章。

我先从网上下载了《海子诗全集》的 PDF（就是西川编的那本，公认质量较好）。然后提取正文，去掉目录、序言、后记等。提取需要安装两个库。

提取出纯文本后，用 jieba 进行分词和统计。最后把结果导入 WordArt 生成词云。第一张词云用海子流传最广的肖像照做底图：
